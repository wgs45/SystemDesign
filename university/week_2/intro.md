# 💻 Information Technology

## 🧠 Concept

**Information Technology (IT)** is the combination of hardware and software used to **store, manage, access, communicate, and share information**.

### Key Ideas

- 🖥️ **Hardware** → physical devices such as computers and servers
- 📦 **Software** → programs that run on hardware
- 🌐 **Services** → tools and platforms enabling communication and data sharing

### ⚡ Important Characteristics

- 🔄 Technology changes **rapidly and continuously**
- 🏢 These advances **transform how businesses operate**
- ☁️ Modern systems integrate **cloud computing, networks, and mobile devices**

---

### 🔁 Mini Recap

- IT = hardware + software + services
- Enables organizations to manage and communicate information
- Rapid innovation drives business change

---

# 🧱 Information Systems Components

## ⚙️ System Concept

A **system** is a **set of related components that work together to produce specific results**.

### 🧩 Basic System Flow

```

Input Data → Processing → Output Information

```

### Key Elements

| Component  | Description                      |
| ---------- | -------------------------------- |
| 📥 Input   | Raw data entering the system     |
| ⚙️ Process | Operations performed on the data |
| 📤 Output  | Information produced             |
| 👥 People  | Users and stakeholders           |
| 🗄️ Data    | Stored information               |

---

### 🚨 Mission-Critical Systems

Systems that are **essential to business operations**.

Example:

- Banking transaction systems
- Airline reservation systems

Failure could cause **major business disruption**.

---

# 🖥️ Hardware and Software

## 🖥️ Hardware

The **physical layer** of an information system.

Examples:

- Computers
- Servers
- Storage devices
- Network equipment

### ⚡ Moore’s Law

> The number of transistors on an integrated circuit chip **doubles roughly every 24 months**, increasing computing power.

---

## 💾 Software

Software **controls hardware and enables tasks**.

### Types of Software

| Type                    | Purpose                              |
| ----------------------- | ------------------------------------ |
| ⚙️ System Software      | Manages hardware (Operating Systems) |
| 🧩 Application Software | Supports daily business activities   |

### Application Software Categories

- 🔄 **Horizontal systems** – used across industries (e.g., accounting)
- 🏭 **Vertical systems** – industry-specific
- 🕰️ **Legacy systems** – older systems still in use

---

### 🔁 Mini Recap

- Hardware = physical components
- Software = programs controlling hardware
- Both are essential to information systems

# 🧠 Modeling Business Operations

## 🏢 Business Profile

A **business profile** provides an overview of a company.

Includes:

- 🎯 Mission
- 🏭 Functions
- 👥 Organization structure
- 📦 Products/services
- 🤝 Customers and suppliers
- ⚔️ Competitors
- ⚠️ Constraints
- 🚀 Future direction

---

## 🔄 Business Processes

A **business process** is a **series of activities that achieve a specific business goal**.

Example flow:

```

Receive Sales Order
↓
Check Customer Status
↓
Verify Credit
↓
Enter Order Data
↓
Completed Sales Order

```

---

## 🧾 Business Process Modeling

Two common methods:

- 📊 **BPM (Business Process Model)**
- 🧩 **BPMN (Business Process Modeling Notation)**

Used to visually represent how business processes work.

# 🏢 Business Information Systems

Organizations use different systems for different needs.

---

## ⚙️ Transaction Processing Systems (TPS)

Processes **daily business transactions**.

Examples:

- Order processing
- Billing
- Payroll

### Example Sales Transaction

1️⃣ Verify customer data
2️⃣ Check credit status
3️⃣ Check inventory
4️⃣ Post accounts receivable
5️⃣ Adjust inventory
6️⃣ Update sales records

---

## 📊 Business Support Systems

Provide information to support decision making.

Examples:

- 📈 **MIS (Management Information Systems)**
- 📡 **RFID tracking systems**

---

## 🧠 Knowledge Management Systems

Systems designed to **store and organize knowledge**.

### Components

- 📚 **Knowledge Base**
  Large database containing information

- 🔍 **Inference Rules**
  Identify patterns and relationships

---

## 🚀 User Productivity Systems

Technology that improves worker efficiency.

Examples:

- 📝 Office tools
- 🤝 Groupware (collaboration software)
- 🤖 Digital assistants

---

## 🔗 Systems Integration

Combines multiple systems into one unified platform.

Includes:

- Transaction processing
- Business support systems
- Knowledge management
- Productivity tools

---

### 🔁 Mini Recap

Major system types:

- TPS → daily operations
- MIS → decision support
- KM → knowledge storage
- Productivity tools → improve efficiency

# 🧠 Systems Development

Systems are built using structured methods.

---

## 🔄 SDLC (Systems Development Life Cycle)

Main stages:

| Stage        | Description                         |
| ------------ | ----------------------------------- |
| 📋 Plan      | Identify problems and opportunities |
| 🔍 Analyze   | Study business requirements         |
| 🧩 Design    | Create system structure             |
| 🛠 Implement | Build and install system            |
| 🔧 Support   | Maintain and improve system         |

---

## 🧱 Other Development Methods

### 🧩 Object-Oriented Analysis

- Combines **data + processes into objects**

### ⚡ Agile Methods

- Develop system **in small iterative steps**
- Frequent user feedback

### 🌀 Spiral Model

- Combines **development and risk analysis**

---

## 🧪 Prototyping

A **prototype** is an early working version of a system.

### Advantages

- Helps users visualize the system
- Improves feedback

### ⚠️ Disadvantages

- Decisions may be made too early
